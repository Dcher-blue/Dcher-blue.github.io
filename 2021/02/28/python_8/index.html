元组

什么是元组

元组是一个有序的，不可改的数据结构(immutable)
l = [5,3,4] #list
n = (5,3,4) #tuple

元组的优点:1.快 2.安全 3.元组可以被称为dict的key
使用元组:1.传参的时候 2.其他不变的情况
